# Installation

## Prerequisites

- Node.js 18+ (for Node.js environments)
- TypeScript 5.0+ (recommended)

## npm

```bash
npm install diagrams-js
```

## yarn

```bash
yarn add diagrams-js
```

## pnpm

```bash
pnpm add diagrams-js
```

## Browser Usage

For browser usage, you can import directly from a CDN:

```html
<script type="module">
  import { Diagram } from "https://esm.sh/diagrams-js";
  // ... your code
</script>
```

## Optional Dependencies

For PNG rendering in Node.js, install `sharp`:

```bash
npm install sharp
```

## TypeScript Configuration

Make sure your `tsconfig.json` includes:

```json
{
  "compilerOptions": {
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "target": "ES2022"
  }
}
```
