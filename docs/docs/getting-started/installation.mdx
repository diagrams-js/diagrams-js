# Installation

Install diagrams-js using your preferred package manager:

## npm

```bash
npm install diagrams-js
```

## yarn

```bash
yarn add diagrams-js
```

## pnpm

```bash
pnpm add diagrams-js
```

## Requirements

- **Node.js**: 18.0.0 or higher
- **TypeScript**: 5.0 or higher (for TypeScript projects)

## Browser Usage

For browser usage, install via npm and import:

```typescript
import { Diagram } from "diagrams-js";
import { EC2 } from "diagrams-js/aws/compute";
```

Or use a CDN:

```html
<script type="module">
  import { Diagram } from "https://cdn.jsdelivr.net/npm/diagrams-js@latest/dist/index.js";
</script>
```

## TypeScript Configuration

Make sure your `tsconfig.json` includes:

```json
{
  "compilerOptions": {
    "module": "NodeNext",
    "moduleResolution": "NodeNext",
    "target": "ES2022"
  }
}
```

## Optional Dependencies

For PNG rendering in Node.js, you may optionally install `sharp`:

```bash
npm install sharp
```
