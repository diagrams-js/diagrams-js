# Providers

ts-diagrams includes 17 providers with over 1000 node classes.

## Usage

```typescript
import { EC2, Lambda } from 'ts-diagrams/providers/aws/compute';
import { RDS } from 'ts-diagrams/providers/aws/database';
import { GCE } from 'ts-diagrams/providers/gcp/compute';
```

## Provider Structure

Each provider is organized by service category:

```
providers/aws/
  ├── compute.ts     # EC2, Lambda, ECS, etc.
  ├── database.ts    # RDS, DynamoDB, etc.
  ├── storage.ts     # S3, EBS, etc.
  └── ...
```
