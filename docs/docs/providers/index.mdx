# Providers

diagrams-js includes 17 providers with over 1000 node classes.

## Available Providers

| Provider          | Import Path                  | Services                     |
| ----------------- | ---------------------------- | ---------------------------- |
| **AWS**           | `diagrams-js/aws/*`          | 500+ services                |
| **Azure**         | `diagrams-js/azure/*`        | 400+ services                |
| **GCP**           | `diagrams-js/gcp/*`          | 100+ services                |
| **Kubernetes**    | `diagrams-js/k8s/*`          | Pods, Deployments, Services  |
| **Alibaba Cloud** | `diagrams-js/alibabacloud/*` | Compute, Database, Network   |
| **DigitalOcean**  | `diagrams-js/digitalocean/*` | Compute, Database, Network   |
| **Elastic**       | `diagrams-js/elastic/*`      | Elasticsearch, Kibana, Beats |
| **Firebase**      | `diagrams-js/firebase/*`     | Develop, Grow, Quality       |
| **Generic**       | `diagrams-js/generic/*`      | VPN, OS, Storage             |
| **GIS**           | `diagrams-js/gis/*`          | Data, Desktop, Mobile        |
| **IBM**           | `diagrams-js/ibm/*`          | Compute, Data, Network       |
| **OCI**           | `diagrams-js/oci/*`          | Oracle Cloud Infrastructure  |
| **On-Premises**   | `diagrams-js/onprem/*`       | Docker, Kubernetes, DBs      |
| **OpenStack**     | `diagrams-js/openstack/*`    | Compute, Networking          |
| **Outscale**      | `diagrams-js/outscale/*`     | Compute, Network             |
| **Programming**   | `diagrams-js/programming/*`  | Languages, Frameworks        |
| **SaaS**          | `diagrams-js/saas/*`         | Third-party services         |

## Usage

```typescript
import { EC2, Lambda } from "diagrams-js/aws/compute";
import { RDS } from "diagrams-js/aws/database";
import { GCE } from "diagrams-js/gcp/compute";
```

## Provider Structure

Each provider is organized by service category:

```
providers/aws/
  ├── compute.ts     # EC2, Lambda, ECS, etc.
  ├── database.ts    # RDS, DynamoDB, etc.
  ├── storage.ts     # S3, EBS, etc.
  └── ...
```
