# Elastic

Node classes list for the Elastic Stack provider.

## Import

```typescript
import { Diagram } from "diagrams-js";
import { Elasticsearch, Logstash, Kibana } from "diagrams-js/elastic/elasticsearch";
import { Beats } from "diagrams-js/elastic/beats";
```

## Available Categories

:::note
This provider supports elasticsearch, beats, and other categories. Please refer to the [Python diagrams Elastic documentation](https://diagrams.mingrammer.com/docs/nodes/elastic) for the complete list of available nodes.
:::

:::tip Complete Reference
For a complete list of all Elastic nodes, see the [Python diagrams Elastic documentation](https://diagrams.mingrammer.com/docs/nodes/elastic). All node classes listed there are available in diagrams-js with the same class names.
:::
