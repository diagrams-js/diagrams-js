# Generic

Node classes list for the Generic computing provider.

## Import

```typescript
import { Diagram } from "diagrams-js";
import { Compute, Rack } from "diagrams-js/generic/compute";
import { Database } from "diagrams-js/generic/database";
import { Firewall, Router, Switch } from "diagrams-js/generic/network";
import { Storage } from "diagrams-js/generic/storage";
```

## Available Categories

:::note
This provider supports compute, database, network, OS, and storage categories. Please refer to the [Python diagrams Generic documentation](https://diagrams.mingrammer.com/docs/nodes/generic) for the complete list of available nodes.
:::

:::tip Complete Reference
For a complete list of all Generic nodes, see the [Python diagrams Generic documentation](https://diagrams.mingrammer.com/docs/nodes/generic). All node classes listed there are available in diagrams-js with the same class names.
:::
