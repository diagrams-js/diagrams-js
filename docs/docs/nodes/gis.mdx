# GIS

Node classes list for the GIS (Geographic Information System) provider.

## Import

```typescript
import { Diagram } from "diagrams-js";
// Import from "diagrams-js/gis/*"
```

## Available Categories

:::note
This provider supports data, desktop, mobile, server, and other categories. Please refer to the [Python diagrams GIS documentation](https://diagrams.mingrammer.com/docs/nodes/gis) for the complete list of available nodes.
:::

:::tip Complete Reference
For a complete list of all GIS nodes, see the [Python diagrams GIS documentation](https://diagrams.mingrammer.com/docs/nodes/gis). All node classes listed there are available in diagrams-js with the same class names.
:::
