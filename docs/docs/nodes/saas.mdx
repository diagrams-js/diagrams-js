# SaaS

Node classes list for the SaaS (Software as a Service) Applications provider.

## Import

```typescript
import { Diagram } from "diagrams-js";
import { Slack, Discord } from "diagrams-js/saas/chat";
import { Datadog, Newrelic } from "diagrams-js/saas/logging";
import { Cloudflare } from "diagrams-js/saas/cdn";
```

## Available Categories

:::note
This provider supports alerting, analytics, CDN, chat, and many other categories. Please refer to the [Python diagrams SaaS documentation](https://diagrams.mingrammer.com/docs/nodes/saas) for the complete list of available nodes.
:::

:::tip Complete Reference
For a complete list of all SaaS nodes, see the [Python diagrams SaaS documentation](https://diagrams.mingrammer.com/docs/nodes/saas). All node classes listed there are available in diagrams-js with the same class names.
:::
